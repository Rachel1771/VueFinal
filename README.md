## Project setup
```
cnpm install
```

### 安装一下后台服务的包
1. node.js服务  
2. express 
3. jwt（生成token）jsonwebtoken   解析token: 安装： jwt-decode 
4. mysql
5. mockjs  -- 模拟数据 
   1. 安装： cnpm i mockjs -S 
   2. 引入： 
      node.js: const Mock = require('mockjs')
      前端js:  import Mock from 'mockjs'

    3. 语法：
       Mock.mock() 
6. swiper
7. bootstrap
8. element
9. echarts
### Run
```
cnpm run serve
```
